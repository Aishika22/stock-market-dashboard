# 📊 Stock Market Data Visualization

This project analyzes stock market data using Data Science techniques and displays results through a web-based dashboard.

## 🚀 Features
- Real-time stock data using yfinance  
- Moving average analysis (MA20, MA50)  
- Graph visualization using Matplotlib  
- Dashboard showing price, high, low, and trend  
- User-friendly interface  

## 🛠️ Technologies Used
- Python  
- Pandas  
- Matplotlib  
- Flask  
- HTML, CSS, JavaScript  

## ▶️ How to Run the Project

1. Install required libraries:pip install -r requirements.txt
